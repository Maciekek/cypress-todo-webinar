## Jak uruchomić testy

- Najpiew pobierz lub sklonuj to repozytorium 
- następnie w tym katalogu wykonak polecenie:  
```bash
    npm install
```
- następnie ucuhom Cypress
``` bash
    npm cypress:open
```
